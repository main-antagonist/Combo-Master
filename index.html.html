📸 Image Performance Tips
For best performance, use JPEG images at a recommended resolution of 250 × 350 px, roughly 50–60 KB per file.
Large or high-resolution images (PNG, full-art scans, etc.) are stored as base64 in the browser and can significantly slow down the tool — especially in the Editor with many steps.
If the tool feels sluggish after uploading, try compressing your images with a tool like Squoosh before re-uploading.

📝 Plain Text Input

Format: combo name on first line, then numbered steps. Optionally add a card name in [brackets] at the start of a step.

My Combo Name

1. [Dark Magician] Normal summon Dark Magician.
2. [Blue-Eyes] Tribute summon Blue-Eyes White Dragon.
3. Activate Pot of Greed and draw 2 cards.
4. [Card A + Card B] Special summon both.

📦 JSON Output / Input

Paste existing JSON here to convert back to text, validate it, or import it directly.

Text Format Guide

Line 1: Combo name (no number prefix)

Steps: Start with a number and period: 1. 2. etc.

Card names: Wrap in square brackets: [Card Name] — multiple: [Card A + Card B]

Interruptions: Add on a new line starting with ! then optionally (low) (medium) (high) (critical)

Example with interruption:

2. [Reinforcement of the Army] Search Elemental HERO Stratos.
! Ash Blossom can negate this (high)

About Combo Master

A portable, offline-capable browser tool for building and sharing card game combo guides. Works with custom cards.

Keyboard Shortcuts

  • ← / → — Navigate steps (Viewer)
  • Ctrl+Enter — Save combo (Editor)
  • Escape — Close modals / picker

🖨 Printing Tips

  • Use the 🖨 Print Combo button in the header to print a clean, card-art-included version of any combo.
  • First select your combo in the Viewer tab, then click Print Combo — it opens a focused print window with just the steps, card images and interruptions.
  • For a quick text-only version, use the ⇄ Flow View in the Viewer tab and use your browser's built-in print (Ctrl+P / Cmd+P).
  • Resize cards from the Viewer tab card size slider before printing — experiment to find the size you prefer.

📸 Custom Card Image Tips

For best performance, use JPEG images at a recommended resolution of 250 × 350 px and roughly 50–60 KB per file. Large or high-resolution images (PNG, full-art scans, etc.) are stored as base64 in the browser and can significantly slow the tool — especially in the Editor with many steps. If the tool feels sluggish after uploading custom cards, try compressing your images with a tool like Squoosh before re-uploading.
